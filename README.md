# FinPilot AI

An AI-powered financial planning platform built with Python.

## Features

- Customer Management
- Pension Forecasting
- Investment Projections
- AI Financial Assistant
- Financial Report Generation
- Modular Python Architecture

## Tech Stack

- Python
- Streamlit
- SQLite
- OpenAI API
- Pandas

## Project Structure

```
app.py
services/
modules/
database.py
models.py
requirements.txt
```

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
